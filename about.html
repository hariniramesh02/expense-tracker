ℹ️ About ExpensePro

ExpensePro is a personal expense tracking web app built with 💖 by Akshu. It’s designed to help you track, visualize, and manage your daily spending — all in your browser, with zero backend.

📦 Features

  • ✅ Add, view, edit, and delete expenses
  • 📁 Manage your own expense categories
  • 📊 Visual reports using Chart.js
  • 👤 Custom profile with budget & theme options
  • 🗃️ Fully localStorage-based (no login needed)
  • 🚀 Responsive, modern UI using Bootstrap 5

🛠️ Tech Stack

Built with: HTML CSS Bootstrap JavaScript Chart.js

📈 Future Plans

  • 🔐 Add user authentication
  • ☁️ Sync data to cloud
  • 📱 Turn it into a mobile app with PWA
